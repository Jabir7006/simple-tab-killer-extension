# Tab Killer 💤

A hyper-lightweight Chrome extension that lets you manually "kill" or sleep browser tabs to completely free up RAM and CPU usage.

Unlike Chrome's native tab discarding feature (which automatically and annoyingly reloads the page the moment you switch back to the tab), Tab Killer replaces the heavy webpage with a pure, static local HTML file. This means the tab stays asleep even when you focus on it, ensuring zero background performance overhead. 

## ✨ Features
* **Zero Resource Usage:** Swaps heavy websites for a microscopic local page, dropping RAM and CPU usage to virtually zero.
* **No Auto-Wake on Focus:** Switching to a sleeping tab will **not** trigger it to load. It stays dead until you say so.
* **Smart Wake-Up:** Want the tab back? Just click your browser's native **Reload** button, and the extension instantly redirects you back to the original page.
* **Minimalist UI:** Features a clean, dark-themed sleep screen that looks great and runs fast.
* **Privacy Focused:** Requires minimal permissions. No data collection, no background tracking.

## 🚀 Installation (Developer Mode)

1. Clone or download this repository to your local machine.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable **"Developer mode"** in the top right corner.
4. Click **"Load unpacked"** and select the directory containing the extension files.
5. Pin the **Tab Killer** icon to your browser toolbar for quick access!

## 🛠️ How to Use
1. When you're on a heavy page (like a YouTube video or complex web app) that you want to suspend, click the Tab Killer extension icon.
2. The tab will instantly switch to the sleep screen, freeing up resources.
3. You can safely switch away from and back to this tab without it reloading.
4. To restore the tab, simply hit your browser's **Reload** button (or click the "Wake Up" button on the screen).
